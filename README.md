# hahn-solution

Todo app based on Vuejs freamwork.

## Features

- Insert tasks by title
- See tasks list
- Make tasks done or undone
- Delete tasks
- Using Vuex
- Typescript support

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```
